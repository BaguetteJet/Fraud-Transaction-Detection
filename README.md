# Fraud Credit Card Transaction Detection
CS4136 Hackathon - Data Innovation Challenge 2026

## Quick Start

### 1. Create virtual environment 

On vscode create a new virtual environment called '.venv' 
   
Press CTRL+' to open terminal and enter ```python -m venv .venv```   
   
Activate virtual environment using ```.venv\Scripts\activate```

Import requiremnets with ```pip install -r requirements.txt```

### 2. Set environment variables

Create a file called '.env' containing your API key

API_KEY='YOUR_KEY_HERE'
